# Contributor Guides

This section contains contributor-focused documentation for developing and
maintaining the `nomad-simulations` schema and docs. It is the process-focused
subsection of the published [Schema Development](../schema_development/overview.md)
section.

## Where to Find What

- [Schema Development Guide](schema_development_guide.md): development workflow, PR expectations, and maintenance practices.
- [Documentation Writing Guide](documentation_writing_guide.md): concise rules for structuring and maintaining hand-written documentation.
- [Documentation Automation Guide](documentation_automation_guide.md): automation commands for generated docs, fragments, snippets, and validation.

For implementation-focused published docs aimed at parser authors and schema
extenders, see [Schema Development](../schema_development/overview.md).
